# 🐞 Practibug

> **Master Debugging. Not Just Coding.**

Practibug is an AI-powered SaaS platform designed to help developers improve their debugging skills through hands-on practice. Instead of writing code from scratch, users solve AI-generated buggy code challenges inspired by real-world software development scenarios.

Whether you're preparing for technical interviews, improving your problem-solving skills, or becoming a better software engineer, Practibug provides an engaging way to practice debugging.

---

## ✨ Features

- 🤖 AI-generated debugging challenges
- 🐛 Realistic production-like bugs
- 🎯 Three difficulty levels
  - Easy
  - Medium
  - Hard

- 💡 Progressive hints and explanations
- 🧠 Improve logical thinking and debugging skills
- 📊 Track progress and performance

---

## 🚀 Why Practibug?

Most coding platforms focus on writing code.

Practibug focuses on something every professional developer does daily:

> **Finding and fixing bugs.**

You'll work with intentionally broken code that mimics real-world issues such as:

- Syntax errors
- Logic bugs
- React state issues
- Async/await mistakes
- API integration problems
- Performance bottlenecks
- Edge-case failures

## 🚧 Project Status

Practibug is currently in the MVP (Minimum Viable Product) and active development stage.

The project is being built incrementally with a focus on delivering a high-quality debugging learning experience for developers. Features, UI, and functionality are actively evolving, and some planned capabilities may not yet be available.
